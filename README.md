# Railway crew scheduling (PuLP)

Kratak primer projekta za rešavanje problema formiranja dužnosti železničke posade koristeći PuLP i CBC solver.

Instalacija:

```bash
python3 -m pip install -r requirements.txt
```

Pokretanje:

```bash
python3 main.py
```

Rezultati analize osetljivosti biće sačuvani u `outputs/sensitivity.csv`, `outputs/sensitivity.png` i dodatnim pojedinačnim PNG grafovima u `outputs/`.
